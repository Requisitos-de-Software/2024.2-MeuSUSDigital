    Ferramentas

    Introdução

    Este documento trata da descrição de diversas ferramentas que foram selecionadas a fim de facilitarem a criação, organização e o controle no desenvolvimento deste projeto. Também foram selecionadas ferramentas essenciais para a comunicação da nossa equipe. Nos baseamos no trabalho desenvolvido por equipes de semestres anteriores da disciplina, como as dos projetos Bilheteria digital e Economia DF. A Tabela 1 apresenta, em ordem alfabética, as ferramentas selecionadas que utilizaremos no desenvolvimento do projeto, acompanhadas de suas respectivas finalidades.

    Tabela 1 – Ferramentas utilizadas no projeto

    Ferramenta Finalidade Etapa Artefato
    diagrams.net (antigo draw.io): Utilizado para criar diagramas e modelos visuais, como o Rich Picture, facilitando o entendimento e o planejamento visual do projeto. [1]. Planejamento e Análise Rich Picture, Diagramas de Fluxo
    Figma: Ferramenta para criação de protótipos de interfaces e layouts de alta fidelidade, proporcionando uma visão gráfica do projeto. [2]. Design de Interface Protótipos de Interface
    Git: Sistema de controle de versões usado para gerenciar e acompanhar as mudanças nos arquivos do projeto. [3]. Desenvolvimento Repositório de Código
    GitHub: Plataforma de hospedagem para o repositório do projeto, facilitando o trabalho colaborativo e o controle de versões. [4]. Desenvolvimento Repositório de Código
    Google Docs: Utilizado para criação e edição de documentos e textos colaborativos, permitindo que a equipe trabalhe em conjunto em tempo real. [5]. Documentação Documentos de Requisitos
    Google Forms: Ferramenta usada para criar questionários e coletar feedback de usuários, auxiliando na avaliação e validação do projeto ou de etapas do mesmo. [6]. Avaliação Questionários de Feedback
    Google Sheets: Utilizado para a organização e acompanhamento de cronogramas, além de análises de dados para o projeto. [7]. Planejamento Cronogramas, Análises de Dados
    Microsoft Teams: Ferramenta de comunicação para reuniões e compartilhamento de arquivos, permitindo a colaboração e interação da equipe. [8]. Comunicação Reuniões, Arquivos Compartilhados
    Miro: Ferramenta para criação de quadros colaborativos e visuais, como mapas mentais e diagramas de fluxo, apoiando o planejamento do projeto. [9]. Planejamento e Análise Mapas Mentais, Diagramas de Fluxo
    MkDocs: Utilizado para construir e organizar a documentação do projeto de maneira estruturada e de fácil navegação. [10]. Documentação Documentação do Projeto
    This Person Does Not Exist: Ferramenta usada para gerar imagens fictícias de pessoas, auxiliando na criação de personas para o projeto. [11]. Análise de Requisitos Personas
    Sony Vegas Pro: Utilizado para edição de vídeos e apresentações, permitindo a criação de materiais audiovisuais para o projeto. [12]. Apresentação Vídeos, Apresentações
    Visual Studio Code: Editor de código para desenvolvimento e edição dos arquivos de documentação e scripts do projeto. [13]. Desenvolvimento Códigos, Scripts
    WhatsApp: Principal canal de comunicação rápida e informal entre os membros da equipe, facilitando o alinhamento de tarefas. [14]. Comunicação Mensagens, Alinhamentos
    YouTube: Utilizado para compartilhar apresentações e disponibilizar gravações de reuniões para consulta da equipe e do professor. [15]. Apresentação Vídeos, Gravações de Reuniões
    ChatGPT: Ferramenta de IA utilizada para auxiliar na geração de ideias, revisão de textos e sugestões de melhorias no projeto. [16]. Análise e Desenvolvimento Ideias, Revisão de Textos
    DeepSeek: Ferramenta de IA utilizada para análise de dados e sugestões de otimização no projeto. [17]. Análise e Desenvolvimento Análise de Dados, Sugestões de Otimização
